## Run and Deploy

This contains everything you need to run your app locally.

### Run Locally

**Prerequisites:** [Node.js](https://nodejs.org/)

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3.  Run the app:
    ```bash
    npm run dev
    ```
